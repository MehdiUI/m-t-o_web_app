🌤️ Weather App
Une application météo moderne et élégante développée avec React 19 et TypeScript. Interface fluide et intuitive avec visualisations graphiques, recherche de villes et thème sombre/clair.

✨ Fonctionnalités principales :

🌡️ Météo en temps réel avec prévisions

📊 Graphiques interactifs (Recharts)

🔍 Recherche intuitive par ville

🌙 Mode sombre/clair automatique

📱 Design responsive parfait

⚡ Performances optimisées (Vite)

🛠 Stack technique : React 19 • TypeScript • Vite • Tailwind CSS • Radix UI • TanStack Query • Recharts
